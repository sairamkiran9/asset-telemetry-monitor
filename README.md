# Asset Telemetry Monitor Microservices 
  
  A simple microservices architecture using gRPC, Go, and Docker.
  
  ## Services
  - Asset Registry (Port 50051)
  - Telemetry Service (Port 50052)
  - Monitoring Service (Port 50053)
  
  ## Prerequisites
  - Go 1.21+
  - Docker Desktop
  - protoc compiler
